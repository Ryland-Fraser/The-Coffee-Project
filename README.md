# The-Coffee-Project

The Coffee Project is a work-in-progress web application focused on helping people discover coffee shops and drinks in the Edmonton area. The goal is to give users a personalized 'coffee fortune' by recommending both a coffee shop and a specific drink, while also allowing users to browse and manage reviews for shops and individual drinks.

This project is mainly being used as a way to learn and practice full stack web development by building a real-world application from the ground up. Development is being done with both desktop and mobile users in mind, with a focus on responsive and accessible design.
